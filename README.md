# miPal

[miPal deployed to Netlify](https://mipal.netlify.app/)

## Description

miPal is a clone of a popular fitness and nutrition tracking application. It allows users to create an account, log in, and track their daily nutrition and fitness goals. Users can also create and join groups to share their progress with others.
